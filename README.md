ws-file input
===========

Styled File input box

Add a file upload box and use this plugin to convert it to a stylable element.

How to use:

Html:
```html
<input class="file-input" name="helloworld" type="file" id="myid" />
```

Jquery:
```html
 $(".file-input").wsFileInput();
```


Defaults:
value : "Browse"
